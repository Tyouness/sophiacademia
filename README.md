# Sophiacademia

Base technique Next.js 14+ avec TypeScript, TailwindCSS et Supabase.

## Prerequis

- Node.js 18+ (ou 20+ recommande)
- Un projet Supabase

## Configuration

1) Copier les variables d'environnement:

```bash
cp .env.example .env.local
```

2) Renseigner les valeurs:

- NEXT_PUBLIC_SUPABASE_URL
- NEXT_PUBLIC_SUPABASE_ANON_KEY
- SUPABASE_SERVICE_ROLE_KEY (serveur uniquement)

## Lancer en local

```bash
npm run dev
```

Ouvrir http://localhost:3000

## Test Supabase

La route /test-db effectue une requete serveur sur la table `healthcheck`.

SQL minimal (a executer dans Supabase) :

```sql
create table if not exists public.healthcheck (
  id bigint generated by default as identity primary key,
  created_at timestamp with time zone default now()
);

insert into public.healthcheck default values;
```
